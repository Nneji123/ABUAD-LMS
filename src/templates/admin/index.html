ABUAD LMS Admin Page

Welcome to the admin page of the ABUAD LMS.

The ABUAD LMS provides a comprehensive learning management system for students, faculty and staff at Afe Babalola University, Ado-Ekiti (ABUAD). The system offers a range of features to support teaching, learning and administration at the university.

Some of the key features of ABUAD LMS include:

  • Facial recognition based attendance system, which helps track attendance in real-time and minimizes the possibility of errors.
  • Video and document viewing and uploading, allowing instructors to share lecture recordings and learning materials with students.
  • Course uploads and exam eligibility with attendance recording, which provides a centralized location for course materials, assignments, and exams.

The ABUAD LMS is designed to be user-friendly and intuitive, with easy navigation and clear instructions for both instructors and students. It also provides a range of tools and resources to help instructors create engaging and interactive learning experiences for their students.
